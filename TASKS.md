## Дорожная карта по внедрению отчетов

### Этап 1. Высокий приоритет

- [ ] Реализовать отчет по продажам по часам
- [ ] Доработать SLA-доставки по этапам приготовления, полки и пути
- [ ] Реализовать отчет по опозданиям доставок
- [ ] Вывести KPI курьеров в отдельный экран
- [ ] Выделить маркетинговые источники в отдельный отчет

### Этап 2. Операционная детализация

- [ ] Реализовать сводку доставки за период
- [ ] Добавить карту курьеров и заказов
- [ ] Доработать отчет по акциям и промокодам
- [ ] Расширить блок меню и ассортимента сверх стоп-листа

### Этап 3. Планирование производства

- [ ] Реализовать прогноз загрузки производства по часам
- [ ] Учесть подтвержденные предзаказы в прогнозе
- [ ] Добавить сравнение с прошлой неделей и аналогичным днем

### Технический контур

- [ ] Проверять OLAP-поля через endpoint колонок перед внедрением каждого отчета
- [ ] Унифицировать сопоставление организаций и подразделений
- [ ] Сохранить кэш, retry и деградацию для нестабильного iiko OLAP
